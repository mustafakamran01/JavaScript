 
 #Types of Events in JavaScript: 
 
 => 'click' : It triggers when we click on something like button, div, paragraph, etc.

 => 'dblclick' : It triggers when we double-click on something like button, div, paragraph etc.

 => 'mouseenter' : It triggers when mouse enter somewhere like button, div, paragraph, etc.

 => 'mouseleave' : It triggers when mouse leave from somewhere like button, div, paragraph, etc.

 => 'keydown' : It triggers when we click any keyboard button in the given space like input-box, body, div, etc.

 => 'input' : It triggers when we type type something on inputbox, div, etc. It captures the value that has been written.

 => 'submit' : It triggers when we submit any form in the browser.

 => 'mousemove' : It triggers when we move mouse continiously in the given area. 
                  It is generally used to captures the co-ordinates of mouse continiously.

 => 'scroll' : It triggers when we scroll windows.

 => 'blur' : It triggers when focus is losses from the given element like input box, div, button, etc.
